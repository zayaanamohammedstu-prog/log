❓ LogGuard Help

Getting Started

  1. Install dependencies: pip install -r requirements.txt
  2. Run the setup wizard: python scripts/setup.py
  3. Start the server: python -m flask --app app/app.py run
  4. Open http://localhost:5000 in your browser and sign in.

Main Dashboard Tabs

TabPurpose
📊 OverviewKPI cards, charts, and the log upload area
🔍 Anomaly ExplorerSortable table of all anomalous IP/hour buckets with ML scores
📋 Evidence & ReportsAttack chain analysis and HTML/PDF report generation
📤 Export & VisualiseAdvanced charts, CSV/JSON export, email & WhatsApp sharing
📜 HistoryAll previous analysis runs with the ability to reload results
⚙️ AdminUser management, audit ledger, and system settings (admin only)

Key Actions

  • Analyse logs — drag-and-drop or click Choose File on the Overview tab, or press "Analyse sample logs" to use the bundled demo.
  • Drill-down detail — click any row in the Anomaly Explorer to see reason codes, feature deviations, and per-model scores.
  • Download report — use the 📥 Download PDF or Download HTML Report buttons in the Evidence & Reports tab.
  • Share report — use the ✉️ Send via Email or 💬 Send via WhatsApp buttons in the Export & Visualise tab.
  • Model retraining — run python scripts/retrain_model.py retrain from the terminal after collecting feedback.
